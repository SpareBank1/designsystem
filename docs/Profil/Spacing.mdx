# Spacing

Luft i og mellom komponentene er en viktig ingrediens i det visuelle designet

## Bruk av spacing
Spacing er viktig for å gi luft i og mellom komponentene.

Vi bruker variabler basert på [8pt grid](https://spec.fm/specifics/8-pt-grid) for å definere spacing. På denne måten sikrer vi en forutsigbar skalering fremfor å overlate det til tilfeldighetene.

## Variabler

Disse Less-variablene kan brukes til å definere `margin` og `padding` i og rundt komponenter og innhold.
