# Bølge

Bølgen er et sentralt element i SpareBank 1s visuelle profil. Vi bruker den som et identitetsbærende og dekorativt innslag,
for eksempel som bakgrunn på websider. Bølgen tilbys nå kun med blå bakgrunn, og kun i context default (altså ikke i accent kontekst).

<Canvas of={WaveStories.Standard} />
<Controls of={WaveStories.Standard} />

## Subtil bakgrunn

Bølgen kommer med enten hvit eller subtle bakgrunn i default kontekst.

<Canvas of={WaveStories.SubtleBackground} />
<Controls of={WaveStories.SubtleBackground} />

## Snu bølgen

I tilfeller der du ønsker å bruke bølgen opp ned kan du sende med med `flip`-propertyen, som vil rotere bølgen 180 grader.

<Canvas of={WaveStories.Flipped} />
<Controls of={WaveStories.Flipped} />

## Plassering

Bølgen kan forskyves horisontalt ved hjelp av `position`-propertyen.

<Canvas of={WaveStories.Position} />
<Controls of={WaveStories.Position} />
