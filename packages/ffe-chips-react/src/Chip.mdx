# Chip

Bruk Chips for elementer der brukeren trenger å filtrere eller organisere innhold.

Skriv klare og tydelige labels slik at brukeren forstår formålet med ulike chips.
Om brukeren kan velge chips må de også kunne avvelge chips. Dette er spesielt viktig for multiselect eller filtreringsløsninger.
