# CardAction (Klikkbare kort)

## Kode

    code={`

## Forhåndsvisning

Kort er ikke klikkbare som standard. Bruk `<CardAction/>` for å lage et klikkbart kort. En `<CardAction/>` kan brukes som en lenke eller en knapp ved hjelp av as.

Som lenke:

<Canvas of={CardActionStories.Standard} />
<Controls of={CardActionStories.Standard} />

Som knapp:

<Canvas of={CardActionStories.AsButton} />

I `Title`:

<Canvas of={CardActionStories.WithinTitle} />
