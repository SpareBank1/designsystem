# IllustrationCard

Illustrasjonskort er kort med en venstrestilt illustrasjon. Den tilbyr alle underkomponenter (CardName, Title, Subtext, Text og CardAction)
og plasserer dem i gitt rekkefølge på høyresiden nedover i kortet (med teksten venstrestilt)
