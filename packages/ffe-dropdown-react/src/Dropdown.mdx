# Dropdown

Dropdown er en vanlig nedtrekksliste som kan brukes man når man har 5-10 valg å velge mellom. Har du mindre enn 5 valg, bruk `RadioButtons` istedenfor.
Har du flere burde du bruke `SearchableDropdown`.
