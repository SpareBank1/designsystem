## Med nedtrekkslister for måned og år

Kalenderen støtter nedtrekkslister for måned- og årsvalg ved å sette `dropdownCaption={true}`. Dette gjør det mulig for brukeren å navigere raskt til spesifikke måneder og år uten å måtte klikke seg gjennom kalenderen steg for steg. 

Dette er spesielt nyttig når brukeren trenger å velge datoer langt frem eller tilbake i tid.
