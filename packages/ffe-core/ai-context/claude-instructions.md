## SB1 FFE Components

Docs for usage: https://sparebank1.github.io/designsystem/?path=/docs

- Components are listed in the left side navigation. For specific components (e.g. PrimaryButton),
  click its label in the side nav to view its documentation and examples.
- Direct doc for PrimaryButton:
  https://sparebank1.github.io/designsystem/?path=/docs/komponenter-buttons-primarybutton--docs
- Usage examples are shown in expandable sections, each with a "show code" button for code samples.
- If no finding direct link to doc, when asked about "how to use", let me know instead of giving wrong info.
- When asked about "how to use" or usage, try to find direct link to doc.
- Use `@sb1/ffe-*` packages for UI consistency
- Icons: `import iconName from '@sb1/ffe-icons/icons/open/300/md/name.svg'`
- Packages/Components: https://github.com/SpareBank1/designsystem/tree/develop/packages
  Packages folder includes components. Each subpackage has two folders: one for styling and one for React logic.
  Example: https://github.com/SpareBank1/designsystem/tree/develop/packages/ffe-accordion
  and https://github.com/SpareBank1/designsystem/tree/develop/packages/ffe-accordion-react
- Documentation: https://sparebank1.github.io/designsystem/
