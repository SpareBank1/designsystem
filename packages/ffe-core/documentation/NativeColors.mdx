# Farger for Native

Pakken `@sb1/ffe-core` inneholder fargetokens som tokens, css, less og js.
For å bruke disse fargene i native er det enkleste å bygge de til den relevante platformen.

Scriptene som brukes for å bygge fargene til native ligger her: https://github.com/SpareBank1/designsystem/tree/develop/packages/ffe-core/scripts/build-colors/mobile
Kom gjerne med forbedringer.

For å bruke scriptet kan en bruke npx som lar en kjøre scripts fra pakker unten å innstallere de. Enten kjør det i root og legg til riktig path. eller kjør det i riktig mappe med path `.`Paths=./android-test/colors

## android

`npx @sb1/ffe-core build-colors platform=android androidPaths=./android-test/colors`

## ios

`npx @sb1/ffe-core build-colors platform=ios iosPaths=./test/ios/colors`

## json

`npx @sb1/ffe-core build-colors platform=json jsonPaths=./test/json/colors`
