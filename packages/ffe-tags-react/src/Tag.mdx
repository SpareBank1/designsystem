# Tags

En Tag er en liten, fleksibel komponent som brukes til å kategorisere, merke eller gi ekstra kontekst til innhold.
Den finnes i 6 fargevarianter; `neutral`, `info`, `success`, `warning`, `critical` og `tip`. Hver farge kommer i
to varianter; `emphasis` og `subtle`.

## Med Ikon
En Tag kan brukes med eller uten ikon. Ikonet sendes inn som en child til Tag komponenten ved bruk av `@sb1/ffe-icons-react` pakken.
Du må installere pakken og importere stylingen og ikonet separat.
