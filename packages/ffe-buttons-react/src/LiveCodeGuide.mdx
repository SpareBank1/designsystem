# 🔥 Live Code Guide for Buttons

Velkommen til den interaktive live code editoren for FFE Button komponenter! Her kan du eksperimentere med alle button-typer i sanntid.

## 🚀 Hvordan bruke Live Code Editor

### Grunnleggende funksjonalitet
- **Editor**: Skriv JSX-kode i venstre panel
- **Preview**: Se resultatet umiddelbart i høyre panel  
- **Templates**: Velg forhåndsdefinerte eksempler fra dropdown-menyen
- **Auto-save**: Koden din lagres automatisk mens du skriver


### Interaktive eksempler
```jsx
function ButtonDemo() {
    const [count, setCount] = useState(0);
    const [loading, setLoading] = useState(false);
    
    const handleAsyncAction = async () => {
        setLoading(true);
        await new Promise(resolve => setTimeout(resolve, 2000));
        setLoading(false);
        setCount(prev => prev + 1);
    };
    
    return (
        <div style={{ display: 'flex', gap: '12px', flexDirection: 'column' }}>
            <p>Antall klikk: {count}</p>
            <div style={{ display: 'flex', gap: '12px', flexWrap: 'wrap' }}>
                <PrimaryButton onClick={() => setCount(prev => prev + 1)}>
                    Klikk meg (+1)
                </PrimaryButton>
                <SecondaryButton 
                    onClick={handleAsyncAction}
                    isLoading={loading}
                >
                    Asynkron handling
                </SecondaryButton>
                <ActionButton onClick={() => setCount(0)}>
                    Reset
                </ActionButton>
            </div>
        </div>
    );
}

<ButtonDemo />
```

### Formulær eksempel
```jsx
function FormExample() {
    const [email, setEmail] = useState('');
    const [submitted, setSubmitted] = useState(false);
    
    const handleSubmit = (e) => {
        e.preventDefault();
        setSubmitted(true);
        setTimeout(() => setSubmitted(false), 2000);
    };
    
    return (
        <form onSubmit={handleSubmit} style={{ 
            display: 'flex', 
            flexDirection: 'column', 
            gap: '16px',
            maxWidth: '300px'
        }}>
            <input
                type="email"
                value={email}
                onChange={(e) => setEmail(e.target.value)}
                placeholder="Din e-post"
                style={{
                    padding: '12px',
                    border: '1px solid var(--ffe-color-border-primary-subtle)',
                    borderRadius: '4px'
                }}
            />
            <div style={{ display: 'flex', gap: '8px' }}>
                <PrimaryButton 
                    type="submit"
                    isDisabled={!email || submitted}
                >
                    {submitted ? 'Sendt!' : 'Send'}
                </PrimaryButton>
                <SecondaryButton 
                    type="button"
                    onClick={() => setEmail('')}
                >
                    Tøm
                </SecondaryButton>
            </div>
        </form>
    );
}

<FormExample />
```

## 📚 Hvor finne Live Code Editorers

### Individuelle komponenter
- **PrimaryButton**: `Komponenter/Buttons/PrimaryButton` → "LiveCodeEditor" story
- **SecondaryButton**: Kan lages ved å følge PrimaryButton mønsteret
- **ActionButton**: Kan lages ved å følge PrimaryButton mønsteret
- **TertiaryButton**: Kan lages ved å følge PrimaryButton mønsteret

### Omfattende editor
- **Alle buttons**: `Komponenter/Buttons/Live Code Editor` → Inkluderer alle button-typer

## 🛠️ Lage egne Live Code Editorers

For å lage en ny live code editor for en button, bruk generator-scriptet:

```bash
npm run generate:livecode ButtonName packages/ffe-buttons-react/src
```

Eller lag manuelt ved å følge dette mønsteret:

```typescript
import { createLiveCodeStory } from '../../../.storybook/shared/LiveCodeEditor';
import { YourButton } from './YourButton';

export const LiveCodeEditor = createLiveCodeStory(YourButton, {
    templates: {
        simple: {
            name: 'Enkelt',
            icon: '📝',
            code: `<YourButton size="md">
    Klikk meg
</YourButton>`
        }
    },
    additionalComponents: {
        YourButton
    },
    defaultTemplate: 'simple',
    title: 'YourButton Live Code Editor'
});
```
